# Installation
- Install Stylus ([Firefox](https://addons.mozilla.org/ru/firefox/addon/styl-us/), [Chrome](https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne)), then open Stylus > Manager.
- Drag the [JSON file](stylus.json) to the window. This file includes both CSS files.
- **OR** press Create, drag CSS file to the window and set "URL in domain: vk.com".
